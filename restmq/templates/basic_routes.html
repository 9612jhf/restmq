RestMQ

Main Routes and Endpoints

  • /: Main Handler (Endpoint for QueryString GET/POST Requests)
  • /q: Queues (Endpoint for REST)
  • /c: Queues (Comet endpoint)
  • /ws: Queues (WebSocket endpoint)
  • /queue: Queues (Endpoint for JSON protocol)
  • /p: Queue Police Handler
  • /j: Queue Info Handler
  • /stats: Queue stats Handler
  • /control: Queue Police Handler (start/stop)
